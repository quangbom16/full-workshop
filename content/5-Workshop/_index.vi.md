# Triển Khai Lên AWS Cloud Bằng Terraform

#### Tổng quan

Phòng lab này cung cấp hướng dẫn thực hành về cách triển khai hạ tầng trên Amazon Web Services (AWS) bằng Terraform. Người học sẽ tìm hiểu cách định nghĩa hạ tầng dưới dạng mã (Infrastructure as Code), quản lý trạng thái cấu hình, và thực thi quy trình tự động để tạo mới, cập nhật, hoặc hủy tài nguyên đám mây một cách nhất quán và có kiểm soát.
