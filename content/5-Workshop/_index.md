# Deploying To AWS Cloud Using Terraform

#### Overview

This lab provides a guided, hands-on introduction to provisioning cloud infrastructure on Amazon Web Services (AWS) using Terraform. Participants learn how to define infrastructure as code, manage configuration state, and execute automated workflows to create, update, and destroy cloud resources in a controlled and repeatable way.
